How to Setup

  1. 1

    Select your provider & auth type, then paste your API key

  2. 2

    Add channels (optional — can be done later)

  3. 3

    Click Run Setup

  4. 4

    Click Copy Gateway Token

  5. 5

    Click Open OpenClaw UI

  6. 6
    In the OpenClaw UI, go to Overview → paste the token in Gateway Token
    i
    OpenClaw now requires you to manually set the gateway token. In the dashboard, go to Overview and paste your copied token into the Gateway Token field.
  7. 7
    Go to Manage Devices → Approve the pairing request
    i
    New browsers need a one-time device approval. After opening the OpenClaw UI, go to Manage Devices and approve the pending request.
  8. 8
    If you added a channel token, click Approve Channel Access and enter the code
    i
    After setup, message your bot on the channel (e.g. send "hey"). It will reply with a pairing code. Enter that code here to grant DM access. If no code appears, try restarting the bot.
  9. 9

    You should now see Health: OK in the OpenClaw UI
